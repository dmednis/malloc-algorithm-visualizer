# Malloc Algorithm Visualizer

This is a simple web app that allows to visualize First fit, Best fit, Worst fit and Next fit memory allocation algorithms.

### Running instructions
* Install Node JS (>= 8.x)
* Clone repository
* Run `npm install`
* Run `npm run`
